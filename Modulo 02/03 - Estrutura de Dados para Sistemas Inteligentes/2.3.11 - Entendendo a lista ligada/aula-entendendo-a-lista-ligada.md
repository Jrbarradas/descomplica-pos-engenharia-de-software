# Entendendo a lista ligada

# Exercícios


## Questão 01
Uma das operações da lista ligada é o Inserir Início. Sobre essa operação:

### Resposta:
- e) é um módulo procedimento da operação inserir no início que recebe como parâmetro um elemento do tipo No a ser inserido e insere este elemento no início da lista ligada

> SOLUÇÃO DO PROFESSOR ✨
>
> Uma das operações de lista ligada é o Inserir Início. Esta operação é um módulo procedimento que recebe como parâmetro um elemento do tipo No que vai ser inserido na lista ligada e insere este elemento na lista ligada


## Questão 02
Uma das operações da lista ligada é o Destruir. Sobre essa operação:

### Resposta:
- b) é um módulo procedimento sem parâmetros da operação destruir que simplesmente destrói uma lista ligada

> SOLUÇÃO DO PROFESSOR ✨
>
> Uma das operações da lista ligada é o Destruir. Esta operação é um módulo procedimento sem parâmetros que simplesmente vai destruir a lista ligada, fazendo o primeiro e o último apontarem para nulo


## Questão 03
Uma das operações da lista ligada é o Lista Vazia que:

### Resposta:
- a) é um módulo função sem parâmetros da operação lista vazia que retorna verdadeiro se a lista estiver vazia e retorna falso se a lista não estiver vazia

> SOLUÇÃO DO PROFESSOR ✨
>
> Uma das operações da lista ligada é a lista vazia. Esta operação é uma função sem parâmetros que vai verificar se a lista ligada possui elementos. Caso tenha elementos, a função vai retornar falso. Caso não tenha elementos, a função vai retornar verdadeiro


## Questão 04
Uma das operações da lista ligada é o Elemento Início que:

### Resposta:
- d) é um módulo procedimento da operação elemento do início que mostra ao usuário o primeiro elemento que está na lista ligada

> SOLUÇÃO DO PROFESSOR ✨
>
> Uma das operações da lista ligada é o elemento do início. Esta operação é um procedimento sem parâmetros que vai mostrar uma cópia do elemento que está no início da lista ligada, sem remover este elemento
