## Questão 01
O Apache Sqoop é uma ferramenta que trata especificamente do quê:

### Resposta:
- c) Ferramenta de ingestão de dados, baseados em dados de bancos relacionais para o HDFS e o HIVE, garantindo a rápida inclusão de dados no Data Lake

> SOLUÇÃO DO PROFESSOR ✨
>
> O SQoop é uma ferramenta de ingestão de dados basicamente consegue copiar dados existentes em bancos de dados relacionais e transportar para o HIVE e ou HDFS de forma rápida e precisa, utiliza o conceito de Map Reduce e todo o cluster Hadoop e a linguagem Java para isso
